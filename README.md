# Kommando

Meine Steuerzentrale für den Heimserver – direkt aufs iPhone. Speicher, Container, VMs,
Freigaben, System & Neustart, dazu **Home Assistant** (Licht, Klima, Sensoren mit Verlaufsgraphen,
Wetter), **Minecraft** und ein **KI-Chat**, der den Server bedienen kann. Alles im **Kinekt**-Look:
tiefes Schwarz, weicher Glow, dünne Schrift – wie [Kino](https://github.com/nicolasjankovich-netizen/kino).

## Was die App kann
- **Server** – Pools/Datasets, Container (Start/Stopp/Logs), VMs, Freigaben (SMB/NFS), SMART, Updates.
- **Zuhause** – Home-Assistant-Dashboard: Räume, Licht+Dimmen, Klima+HVAC, Media, Sensoren mit 24-h-Charts, Wetter.
- **Minecraft** – Server-Status & Steuerung.
- **KI** – Chat, der Server-Aktionen ausführt (Face-ID für Kritisches).

## Installieren (SideStore)
1. **SideStore** auf dem iPhone einrichten (einmalig).
2. In SideStore → **Sources → +**:
   `https://raw.githubusercontent.com/nicolasjankovich-netizen/kommando/main/source/apps.json`
3. Kommando antippen → **Get/Install**. Erneuert sich automatisch über WLAN.
